VotePoint: Secure Console Voting

VotePoint is a robust C++ Command-Line Application designed to manage digital elections. It bridges the gap between simplicity and functionality by providing a secure environment for both administrators and voters.

The core logic uses File Handling (fstream) to ensure data persistence, meaning all votes, registered voters, and topics are saved permanently.
